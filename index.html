Exoplanets

An app for displaying information about confirmed exoplanets and the stars they orbit. Uses the API provided by NASA's Exoplanet Science Institute to show some basic current information about known exoplanets. Includes names, distances, discovery methods and simple visuals to illustrate sizes.

Python, Flask, Azure, Jade, D3, JavaScript, Foundation

Election Results

A small Rails application for dispalying results from the last Canadian Federal election. Particularly interested in the margin of victory for each riding and the consequences of the first-past-the-post system. Also includes charts for each riding's vote talley created with D3js.

Ruby, Rails, HAML, Heroku, D3, JavaScript, Foundation

Accessible Parking

A simple Android app for finding accessible parking spaces in Vancouver. Uses open data provided by the City of Vancouver including the location and description of available spaces. I used essentially the same code for another app that shows nearby drinking fountains.

Ionic, Angular, JavaScript, Google Maps API

X-ReadingList

A small project for finding comic books using possibly excessively detailed criteria. For example, all issues of X-Men or Uncanny X-men between 1990 and 2010 featuring Rogue and Mystique or Destiny, but excluding anything pencilled by Greg Land. So yeah, possibly overly detailed.

Provides issue summaries, credits and (where available) covers and links to buy/read digital copies from Marvel.

Angular, JavaScript, MongoDB, Node, Express, Bootstrap, jQuery
